Explicación del Código

Este programa en Java invierte una palabra utilizando recursividad. La palabra utilizada en este ejemplo es "ejemplo". La función recursiva toma la palabra, la descompone carácter por carácter, y luego reconstruye la palabra en orden inverso.

Clase Recursividad


public class Recursividad {

    // Método recursivo para invertir una palabra
    static String invierte(String palabra) {
        // Caso base: si la palabra está vacía, regresa la palabra tal cual
        if (palabra.isEmpty()) {
            return palabra;
        } else {
            // Llama recursivamente al método con la subcadena y añade el primer carácter al final
            return invierte(palabra.substring(1)) + palabra.charAt(0);
        }
    }

    public static void main(String[] args) {
        // Define una palabra específica
        String palabra = "ejemplo"; // Define la palabra aquí

        // Muestra la palabra invertida en la consola
        System.out.println("Palabra invertida: " + Recursividad.invierte(palabra));
    }
}

Explicación

La clase Recursividad contiene dos métodos principales:

  • invierte(String palabra): Este método recursivo toma una palabra y la invierte. Si la palabra está vacía, regresa la palabra tal como está; de lo contrario, se llama a sí mismo para procesar el resto de la palabra, añadiendo el primer carácter al final.
  • main(String[] args): Este método es el punto de entrada del programa, donde se define la palabra "ejemplo" y se imprime la palabra invertida en la consola.

Método invierte

El método invierte sigue estos pasos:

  1. Comprueba si la palabra está vacía. Si es así, devuelve la palabra.
  2. Si no está vacía, llama recursivamente a invierte con la subcadena que comienza desde el segundo carácter.
  3. Añade el primer carácter de la palabra original al final del resultado recursivo, invirtiendo así la palabra.

Método main

El método main realiza lo siguiente:

  1. Define la palabra "ejemplo" como un ejemplo a invertir.
  2. Llama al método invierte para invertir la palabra.
  3. Muestra el resultado en la consola.
