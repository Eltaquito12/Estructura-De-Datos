Explicación del Código

Este programa en Java calcula el Máximo Común Divisor (MCD) de dos números utilizando un enfoque recursivo basado en el algoritmo de Euclides.

Clase MaximoComunDivisor


import java.util.Scanner;

public class MaximoComunDivisor {
    public static int mcd(int digito, int digito2) {
        if (digito2 == 0) {
            return digito;
        }
        return mcd(digito2, digito % digito2);
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Introduce el primer número: ");
        int digito = scanner.nextInt();

        System.out.print("Introduce el segundo número: ");
        int digito2 = scanner.nextInt();

        System.out.println("El MCD de " + digito + " y " + digito2 + " es: " + mcd(digito, digito2));
        
        scanner.close();
    }
}

Explicación

La clase MaximoComunDivisor contiene dos métodos principales:

  • mcd(int digito, int digito2): Este método recursivo calcula el MCD de dos números utilizando el algoritmo de Euclides.
  • main(String[] args): Este método es el punto de entrada del programa. Solicita al usuario que ingrese dos números, y luego calcula y muestra el MCD de esos números utilizando el método mcd.

Método mcd

El método mcd sigue estos pasos:

  1. Comprueba si el segundo número (digito2) es igual a 0. Si es así, retorna el primer número (digito) como el MCD, ya que esto indica que el MCD ha sido encontrado.
  2. Si digito2 no es 0, el método se llama a sí mismo recursivamente, intercambiando los valores de digito y digito2, y pasando como nuevo argumento digito % digito2 (el residuo de la división).

Método main

El método main realiza lo siguiente:

  1. Solicita al usuario que ingrese dos números.
  2. Llama al método mcd con esos dos números para calcular su MCD.
  3. Muestra el resultado del MCD en la consola.
