Explicación del Código

El siguiente código en Java suma los dígitos de un número de manera recursiva, es decir, descompone el número en sus dígitos y suma cada uno.

Clase examen


package unidad2;

public class examen{
	
	// Elaborar un programa que sume los dígitos de un número de forma recursiva

	public static int sumaDigitos(int numero) {
		if (numero == 0) {
			return 0;
		}
		return numero % 10 + sumaDigitos(numero / 10);
	}

	public static void main(String[] args) {
		int numero = 123;
		int resultado = sumaDigitos(numero);
		System.out.println("La suma de los dígitos de " + numero + " es: " + resultado);
		
		int numero1 = 3457;
		int resultado1 = sumaDigitos(numero1);
		System.out.println("La suma de los dígitos de " + numero1 + " es: " + resultado1);
		
		int numero2 = 987654321;
		int resultado2 = sumaDigitos(numero2);
		System.out.println("La suma de los dígitos de " + numero2 + " es: " + resultado2);
		
		int numero3 = 999999999;
		int resultado3 = sumaDigitos(numero3);
		System.out.println("La suma de los dígitos de " + numero3 + " es: " + resultado3);
	}
}

Explicación

La clase examen contiene dos métodos principales:

  • sumaDigitos(int numero): Este método recursivo toma un número y suma sus dígitos. Lo hace extrayendo el último dígito y llamándose recursivamente con el número restante.
  • main(String[] args): Este método es el punto de entrada del programa, donde se prueban diferentes números para sumar sus dígitos y mostrar el resultado.

Método sumaDigitos

El método sumaDigitos sigue estos pasos:

  1. Si el número es igual a 0, devuelve 0 (caso base).
  2. De lo contrario, toma el último dígito del número usando el operador % 10 y lo suma al resultado de una llamada recursiva que procesa el resto del número (numero / 10).

Método main

El método main realiza lo siguiente:

  1. Define varios números, como 123, 3457, 987654321 y 999999999.
  2. Para cada número, invoca el método sumaDigitos y muestra el resultado en la consola.
