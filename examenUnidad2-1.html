Explicación del Código

Este programa en Java suma los dígitos de un número ingresado por el usuario utilizando un método recursivo. El método procesa cada dígito del número y los suma hasta que no quedan más dígitos.

Clase examen


import java.util.Scanner;

public class examen {
    
    // Elaborar un programa que sume los dígitos de un número de forma recursiva
    public static int sumaDigitos(int numero) {
        // Caso base: si el número es 0, retorna 0
        if (numero == 0) {
            return 0;
        }
        // Suma el último dígito y llama recursivamente al método con el número sin el último dígito
        return numero % 10 + sumaDigitos(numero / 10);
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Solicita al usuario que ingrese varios números y calcula la suma de los dígitos
        System.out.print("Introduce un número: ");
        int numero = scanner.nextInt();
        int resultado = sumaDigitos(numero);
        System.out.println("La suma de los dígitos de " + numero + " es: " + resultado);

        System.out.print("Introduce otro número: ");
        int numero1 = scanner.nextInt();
        int resultado1 = sumaDigitos(numero1);
        System.out.println("La suma de los dígitos de " + numero1 + " es: " + resultado1);

        System.out.print("Introduce otro número: ");
        int numero2 = scanner.nextInt();
        int resultado2 = sumaDigitos(numero2);
        System.out.println("La suma de los dígitos de " + numero2 + " es: " + resultado2);

        System.out.print("Introduce otro número: ");
        int numero3 = scanner.nextInt();
        int resultado3 = sumaDigitos(numero3);
        System.out.println("La suma de los dígitos de " + numero3 + " es: " + resultado3);

        scanner.close();
    }
}

Explicación

La clase examen contiene dos métodos principales:

  • sumaDigitos(int numero): Este método recursivo suma los dígitos de un número. Recibe el número como argumento y procesa su último dígito sumándolo al resultado de la llamada recursiva con el número dividido entre 10.
  • main(String[] args): Solicita al usuario que introduzca varios números y utiliza el método sumaDigitos para calcular y mostrar la suma de los dígitos de cada número.

Método sumaDigitos

El método sumaDigitos sigue estos pasos:

  1. Si el número es 0, retorna 0 (caso base).
  2. Calcula el último dígito del número utilizando el operador módulo (%).
  3. Llama recursivamente al método con el número dividido entre 10, eliminando el último dígito.
  4. Suma el dígito actual al resultado de la llamada recursiva.

Método main

El método main realiza lo siguiente:

  1. Pide al usuario que introduzca varios números utilizando un objeto Scanner.
  2. Llama al método sumaDigitos para cada número ingresado.
  3. Muestra la suma de los dígitos de cada número en la consola.
