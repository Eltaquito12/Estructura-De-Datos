Explicación del Código

Este programa en Java verifica si una palabra predefinida es un palíndromo. La palabra utilizada es "Anita lava la tina", que es un clásico ejemplo de palíndromo. El programa elimina los espacios, convierte la palabra a minúsculas y la evalúa recursivamente.

Clase Palindromo


public class Palindromo {
    
    // Método recursivo para verificar si una palabra es un palíndromo
    static boolean esPalindromo(String palabra) {
        // Caso base: si la palabra tiene 1 o 0 caracteres, es un palíndromo
        if (palabra.length() <= 1) {
            return true;
        }
        
        // Verifica si el primer y último carácter son iguales
        if (palabra.charAt(0) == palabra.charAt(palabra.length() - 1)) {
            // Llama recursivamente al método con la subcadena sin el primer y último carácter
            return esPalindromo(palabra.substring(1, palabra.length() - 1));
        }
   
        // Si los caracteres no coinciden, no es un palíndromo
        return false;
    }

    public static void main(String[] args) {
        // Define una palabra específica
        String palabra = "Anita lava la tina";

        // Elimina espacios y convierte la palabra a minúsculas
        palabra = palabra.replaceAll("\\s+", "").toLowerCase();

        // Verifica si la palabra es un palíndromo y muestra el resultado
        if (esPalindromo(palabra)) {
            System.out.println("La palabra es un palíndromo.");
        } else {
            System.out.println("La palabra no es un palíndromo.");
        }
    }
}

Explicación

La clase Palindromo contiene dos métodos principales:

  • esPalindromo(String palabra): Este método recursivo verifica si una palabra es un palíndromo, eliminando espacios y diferencias de mayúsculas/minúsculas.
  • main(String[] args): Este método es el punto de entrada del programa, donde se evalúa la palabra predefinida "Anita lava la tina".

Método esPalindromo

El método esPalindromo sigue estos pasos:

  1. Comprueba si la palabra tiene uno o cero caracteres. Si es así, se considera un palíndromo.
  2. Verifica si el primer y el último carácter de la palabra son iguales.
  3. Si son iguales, llama recursivamente al mismo método con la palabra sin el primer y último carácter.
  4. Si los caracteres no coinciden, la palabra no es un palíndromo.

Método main

El método main realiza lo siguiente:

  1. Define la palabra "Anita lava la tina" como un ejemplo de palíndromo.
  2. Elimina los espacios en blanco y convierte la palabra a minúsculas para facilitar la comparación.
  3. Verifica si la palabra es un palíndromo utilizando el método esPalindromo.
  4. Muestra el resultado en la consola.
